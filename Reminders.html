Reminders

How it works

Users click on the '+' button to create a new reminder. The screen that opens will give users the option to create a new journal entry or a reminder. Reminders can be customised so that the user can categorise it with different colours. These colours show up on the calendar to easily identify the reminder that the user has put in. When the user opens a specific date on the calendar, they are given their reminder and the option to open their journal. Once the reminder is completed, users can click 'Complete' for it to be done. Alternatively, if the user has not finished the task yet, they can click 'Decline' and the reminder will stay there. The most urgent (closest due date) reminder will show on the user's home screen so that they can quickly access it when opening the application.

The example shown below is of a student who is adding a reminder in for one of their classes. They go to the '+' icon and then they are brought up to a screen where they acn choose to enter in a journal or a reminder. The user is to click on reminder and then fill in the details of the new screen that shows up. The reminder can then be seen in the calendar if the user clicks onto the specified date.

Criteria 1: User is able to navigate to add a reminder to their calendar.
If the user is able to navigate to add a reminder, then Criteria 1 will be met.

Criteria 2: User is able to open up their reminder.
If the user is able to open up their reminder, then criteria 2 will be met.

Criteria 3: User is able to complete their reminder.
If the user is able to complete their reminder and complete it, then criteria 3 will be met.
