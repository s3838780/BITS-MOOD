Coping Strategies

How it works

Users press the button with three lines on the bottom right corner of the screen and then press on the button labeled 'Coping Strategies' to access the main 'Coping Strategies' screen. From there, users can access information directly related to specific mental health conditions such as anxiety or panic attacks. The system would provides users with a variety of information such as, how these can issues can manifest, what can be done, how the users can best cope with the issues, mental health professions the user can contact to discuss the issues, or where the user can read more about these conditions.

This feature will be deemed to meet validation criteria if:

- There is good information on the conditions available to the user

- There are a wide variety of conditions the user can read up on

- There are links to external pages for the user to read more detailed information
